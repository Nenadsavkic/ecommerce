
   PAGINACIJA

Ok, prva beleska na skoro pola kursa. Nova stvar koju smo naucili je Paginacija.
Imamo na primer 24 proizvoda u bazi, zelimo da prikazemo po 8 proizvoda po stranici, 
toznaci da cemo imati 3 stranice po 8 proizvode na svakoj, e sad kako to lepo prikazati
na sajtu, tu dolazimo do paginacije.


Prvo iedmo u 'ProjectController.php' u metodi 'products' vrsomo paginacij

 public function products()
    {
        $products = Product::paginate(8); // paginate(8) -- uzima sve proizvode iz baze i sortira
                                                            vratice 8 proizvoda po strani

        return  view('products', ['products'=> $products]);
    }


Zatim drugi korak je da prikazemo te proizvode i da dobijemo stranicenje, odnosno paginaciju
na html stranici, one brojeve pri dnu koji pokazuju koliko stranica sa proizvodima imamo.
Tu dolazimo do laravel koda koji ce nam napraviti paginaciju na stranici automatski, sa tim 
brjevima stranica.


    <style type="text/css">
         svg{
           width:20px;
         }
         nav p{
           display: none;
         }
         nav .flex a{
           display: none
         }
    </style>
    

    Ovo ispod je taj laravel kod koji kreira paginaciju, pravi nam tri strnice, a Ovo
    gore je css kod koji ispravlja neke greske u prikazu. (Pogledati 'products.blade.php' pri dnu).
    Inace ovaj css kod mozemo prebaciti u 'tooplate-main.css' koji je glavni css fajl ovog projekta,
    u njemu cemo pisati sav css kod.


      <div class="text-center">
        {!! $products->links() !!}
      </div>
