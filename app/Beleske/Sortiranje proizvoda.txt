
SORTIRANJE PROIZVODA


Sortiranje cemo vrsiti na neke od sledecih nacina:

All products  --- svi proizvodi,

Newest  ---  Najnoviji,

Low price  --- Najjeftiniji

High price --- Najskuplji 

Man       ---  Muska odeca

Women     ---  Zenska odeca


Ovo za sve proizvode smo vec uradili, sada idemo dalje.


Newest ili najnovije proizvode cemo doboti tako sto cemo ih sortirati po id-u, 
odnosno proizvodi sa najvecim id-em su poslednji uneti, znaci najnoviji, pa cemo
to iskoristiti da ih sortiramo. idemo u 'ProductController.php' u metodu 'newest'.


   public function newest()
    {
        $products = Product::orderBy('id', 'desc')->paginate(6);
        
        return view('products', ['products' => $products]);
    }


Zatim po ceni od najnize do najvise cene.


      public function lowestPrice()
    {
        $products = Product::orderBy('price', 'asc')->paginate(6);
        
        return view('products', ['products' => $products]);
    }


Zatim od najvise do najnize cene, dakle obrnuto nego u prethodnom primeru.


   public function highestPrice()
    {
        $products = Product::orderBy('price', 'desc')->paginate(6);
        
        return view('products', ['products' => $products]);
    }


Muska odeca.


   public function menClothing()
    {
        $products = Product::where('type', 'men')->paginate(6);
        
        return view('products', ['products' => $products]); 
    }


Zenska odeca.


   public function womenClothing()
    {
        $products = Product::where('type', 'women')->paginate(6);
        
        return view('products', ['products' => $products]); 
    }



Kao sto smo videli gore, u svakoj metodi koristimo istoimenu varijablu '$products', koja
ima drugacije podatke u svakoj metodi, sadrzaj podataka se menja na klik linka, odnosno na 
pozivanje funkciije, to je jos jedna draz Laravela.
